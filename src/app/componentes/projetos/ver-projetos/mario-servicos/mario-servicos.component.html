Mario Bros - Serviços

Este projeto foi desenvolvido juntamente com o instrutor Rodolfo Mori, do curso de Programação Full Stack da instituição de ensino DevClub, com o obejtivo de fixar os conhecimentos em HTML e CSS.

Tecnologias Utilizadas:

  • HTML
  • CSS

Objetivo:

O objetivo deste projeto foi criar uma interface interativa e simples para que o usuário consiga agendar serviços ou consultar os serviços prestados pelos irmãos mais famosos e fofos Mario e Luigi!
