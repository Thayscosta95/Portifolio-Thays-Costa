JokenPô

Este projeto foi desenvolvido juntamente com o instrutor Rodolfo Mori, do curso de Programação Full Stack da instituição de ensino DevClub, com o obejtivo de fixar os conhecimentos em JavaScript. Esse projeto apresenta uma interface moderna, responsiva e com foco em experiência do usuário.

Tecnologias Utilizadas:

  • HTML
  • CSS
  • JavaScript

Objetivo:

O objetivo deste projeto foi criar um jogo divertido e tecnológico de JokenPô, onde o usuário pode jogar contra o computador.
