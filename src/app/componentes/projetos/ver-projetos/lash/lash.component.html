Lash Designer

Este projeto foi uma ideia pessoal, há um tempo atrás eu trabalhei como Lash designer e foi um período de muito aprendizado, inspirada no meu antigo trabalho, quis fazer uma agenda para facilitar os horários marcados dos clientes. Ao abrir a aplicação terá informações sobre o trabalho realizado, a história de como surgiu essa ideia de ser lash designer, e também uma agenda para agendar os horários dos clientes.

Tecnologias Utilizadas:

  • HTML
  • CSS
  • JavaScript

Objetivo:

O objetivo deste projeto foi criar um site de apresentação e juntamente a possibilidade de agendar um horário.
