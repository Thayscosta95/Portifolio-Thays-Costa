Memoteca

Este projeto foi desenvolvido juntamente com a Instrutora Nay da instituição Alura do curso de Angular, com o intuíto de criar um CRUD, com o obejtivo de fixar os conhecimentos em Angular e TypeScript. Esse projeto apresenta uma interface moderna, responsiva e com foco em experiência do usuário.

Tecnologias Utilizadas:

  • HTML
  • CSS
  • TypeScript
  • Angular
  • Node.js

Objetivo:

O objetivo deste projeto foi criar um ambiente onde o usuário tenha a experiencia de criar pensamentos, frases motivacionais e até mesmo lembretes, foi criado um banco de dados fake para guardar esses pensamentos criados, tendo a possibilidade de conultar, editar e excluir.
